{% extends "base.html" %} {% block content %} Pneumonia is a lung infection which causes the inflammation of the small air sacs i.e, Alveoli in one or both lungs. It impacts seven percent of the world’s population, contributing to 2 million children’s deaths per year out of which India contributes about 23% of the global pneumonia burden. The WHO estimates that pneumonia is responsible for one out of every three deaths in India. It is normally detected by a highly-trained consultant’s examination of a chest X-Ray radiograph which is a difficult task even for a professional radiologist. As a result, it is essential to create an automated method for detecting pneumonia to improve diagnosis accuracy. To address this, we developed a deep Transfer learning-based model using complex Visual Group Geometry VGG16 architecture to detect the presence of pneumonia. We analyzed our model with different optimizers to observe any changes in the results. From the result, we can conclude that our model can be used to diagnose pneumonia quickly with satisfactory performance accuracy.

NOTES
  • This deep learning model has been trained and validated, using the following dataset: PAUL MOONEY,Chest X-Ray Images (Pneumonia)
  • The suitability of this deep learning model for clinical use has not been validated. Therefore it cannot be used for any clinical purpose! It can be used for research purposes and for comparison with other similar models.
